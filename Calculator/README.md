\# Calculator



A command-line calculator built in Java that handles basic to intermediate arithmetic operations.



\## What it does

Takes two numbers and an operator as input, then performs the corresponding calculation. Includes division-by-zero handling and power calculations using `Math.pow()`.



\## Supported Operations

\- Addition `+`

\- Subtraction `-`

\- Multiplication `\*`

\- Division `/` (with zero-check)

\- Exponentiation `^`



\## Concepts Used

\- `Scanner` for user input

\- `switch` expressions (Java 14+ arrow syntax)

\- Input validation with boolean flags

\- `Math.pow()` for power operations



\## How to Run

```bash

javac Calculator.java

java Main

```
