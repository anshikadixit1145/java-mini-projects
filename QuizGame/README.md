\# Quiz Game



A multiple-choice Java quiz game with computer science trivia questions.



\## What it does

Presents 5 computer science questions, each with 4 options. The player enters the number of their chosen answer, receives instant feedback, and gets a final score at the end.



\## Questions Cover

\- Networking (routers)

\- Hardware (CPU)

\- Tech history (Facebook launch year, Charles Babbage)

\- Programming history (Fortran)



\## Concepts Used

\- Parallel arrays — `questions\[]`, `options\[]\[]`, `answers\[]`

\- Nested `for` loops for displaying questions and options

\- Score tracking

\- `Scanner` for integer input



\## How to Run

```bash

javac QuizGame.java

java Main

```
