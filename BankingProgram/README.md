\# Banking Program



A menu-driven Java banking application that simulates basic account operations.



\## What it does

Provides an interactive banking menu where the user can check their balance, deposit money, or withdraw funds. Handles edge cases like insufficient funds and negative amounts.



\## Features

\- Check account balance

\- Deposit funds (with negative amount validation)

\- Withdraw funds (with insufficient funds check)

\- Clean exit with goodbye message



\## Concepts Used

\- Static methods — `showBalance()`, `deposit()`, `withdraw()`

\- `switch` expressions with arrow syntax

\- `while` loop for continuous menu interaction

\- `printf` for formatted currency display

\- Input validation and edge case handling



\## How to Run

```bash

javac BankingProgram.java

java Main

```
