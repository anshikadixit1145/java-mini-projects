\# Shopping Cart



A simple Java shopping cart program that calculates the total cost of a purchase.



\## What it does

Asks the user for an item name, price per unit, and quantity, then calculates and displays the total amount due.



\## Concepts Used

\- Mixed data types — `String`, `double`, `int`, `char`

\- Basic arithmetic for total calculation

\- `scanner.nextLine()` and `scanner.nextDouble()` for different input types

\- String concatenation for the receipt display



\## How to Run

```bash

javac ShoppingCart.java

java Main

```



\## Example

```

What item would you like to buy? Headphones

What is the price for each?: 29.99

How many would you like?: 2



You have bought 2 Headphones/s

Your total is $59.98

```
