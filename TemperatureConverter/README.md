\# Temperature Converter



A concise Java program that converts temperatures between Celsius and Fahrenheit using the ternary operator.



\## What it does

Prompts the user to enter a temperature and their desired output unit, then instantly converts and displays the result formatted to one decimal place.



\## Concepts Used

\- Ternary operator `(condition) ? true : false`

\- Method chaining — `scanner.next().toUpperCase()`

\- `printf` for formatted output

\- `Scanner` for user input



\## How to Run

```bash

javac TemperatureConverter.java

java Main

```



\## Example

```

Enter the temperature: 100

Convert to Celsius or Fahrenheit? (C or F): C

37.8° C

```
