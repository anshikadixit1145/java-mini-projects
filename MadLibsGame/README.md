\# Mad Libs Game



A fun word-substitution game in Java where user inputs are plugged into a pre-written story to create silly, unpredictable results.



\## What it does

Prompts the user to enter adjectives, a noun, and a verb, then inserts them into a story template and prints the final result.



\## Concepts Used

\- `String` variables for storing user input

\- `scanner.nextLine()` for reading full-line input

\- String concatenation for building the final story

\- Sequential user prompts



\## How to Run

```bash

javac MadLibsGame.java

java Main

```



\## Example

```

Enter an adjective: enormous

Enter a noun (animal or person): penguin

Enter an adjective: grumpy

Enter a verb ending with -ing: breakdancing

Enter an adjective: confused



Today I went to an enormous zoo.

In an exhibit, I saw a penguin.

penguin was grumpy and breakdancing!

I was confused!

```
