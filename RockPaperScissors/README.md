\# Rock Paper Scissors



A fully playable Rock Paper Scissors game in Java against the computer with replay functionality.



\## What it does

Takes the player's move as text input, generates a random computer move, determines the winner, and asks if the player wants to play again.



\## Features

\- Input validation for invalid moves

\- Computer picks randomly from `{"rock", "paper", "scissors"}`

\- Win / loss / tie detection

\- Replay loop using `do-while`



\## Concepts Used

\- `Random` for computer move selection

\- `do-while` loop for replay

\- String comparison with `.equals()`

\- Input validation using `continue`

\- `.toLowerCase()` for case-insensitive input



\## How to Run

```bash

javac RockPaperScissors.java

java Main

```
