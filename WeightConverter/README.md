\# Weight Converter



A simple Java program that converts weight between pounds (lbs) and kilograms (kgs).



\## What it does

Presents the user with a menu to choose a conversion direction, takes the weight as input, and outputs the converted value formatted to two decimal places.



\## Concepts Used

\- `if-else` conditionals for menu handling

\- `printf` for formatted decimal output

\- Conversion constants (`0.453592` for lbs→kg, `2.20462` for kg→lbs)

\- Input validation for invalid menu choices



\## How to Run

```bash

javac WeightConverter.java

java Main

```



\## Example

```

1: Convert lbs to kgs

2: Convert kgs to lbs

Choose an option: 1

Enter the weight in lbs: 150

The new weight in kgs is: 68.04

```
